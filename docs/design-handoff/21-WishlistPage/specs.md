## 🧩 주요 컴포넌트

### 1. 찜 상품 카드
- **구성**: 상품 이미지 + 정보 + 삭제 버튼 + 담기 버튼
- **Hover**: 이미지 확대, 버튼 등장 (Fade in)
- **삭제 버튼**: 우상단 위치, 클릭 시 목록에서 제거
- **담기 버튼**: 하단 중앙 위치, 장바구니 추가

### 2. 헤더 섹션
- **타이틀**: 하트 아이콘 + "찜한 상품" 텍스트
- **전체 삭제**: 우측 텍스트 버튼 (Red)

### 3. 추천 상품 섹션
- **위치**: 찜 리스트 하단
- **구성**: 연관 상품 4개 그리드
